Web page copy + blogging

To Read, or Not to Read? Book review blog

To Read, or Not to Read?

The brief: My personal project 'To Read, or Not to Read?' is an informative, slightly satirical book review blog. I noticed the internet is saturated with verbose book reviews written by people too concerned with showcasing their intellect. My mission is to get to the point so that readers know quickly if a book is worth reading. To make the site I:

  • Built the website using Wordpress and edited the CSS and HTML code where necessary.
  • Wrote all pages, web copy, reviews and articles.
  • Created branding, aesthetics and brand voice
  • Created graphics for blog posts using Canva and Photoshop
